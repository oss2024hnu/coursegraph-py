This file defines the basic form of data yaml file used in the coursegrap-py project.
이 파일은 coursegrap-py 프로젝트에서 사용하는 data yaml 파일의 기본 형태를 지정합니다.

@YAML 파일의 기본 형식

기본 데이터 정의 (map)
key: value
배열 정의 (array)
key:
  - value
  - value
참/거짓 (boolean)
key: yes or no
key:
  - true
  - false
주석 표현 (Comment)
key: value #Comment

===============================================
@coursegrap-py 의 기본 YAML 형식
(@중요! kye: <value를 공백으로 둘 시 blank(스페이스바 빈칸)가 있지 않도록 주의> ) 

Default key (필수로 지정되어져야하는 키)
과목:
  - 학년: (int) (숫자만 입력) ex) 1
  - 학기: (int) (숫자만 입력) ex) 2
  - 과목명: (str) (문자열로 입력)) ex) 웹스크립트프로그래밍

option key (옵션으로 지정되는 키) 
  - 트랙:
    - (str) (문자열로 입력)
  - 마이크로디그리:
    - (str) (문자열로 입력)
  - 선수과목
    - (str) (문자열로 입력)
