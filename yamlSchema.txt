This file defines the basic form of data yaml file used in the coursegrap-py project.
이 파일은 coursegrap-py 프로젝트에서 사용하는 data yaml 파일의 기본 형태를 지정합니다.

@YAML 파일의 기본 형식

기본 데이터 정의 (map)
key: value
배열 정의 (array)
key:
  - value
  - value
참/거짓 (boolean)
key: yes or no
key:
  - true
  - false
주석 표현 (Comment)
key: value #Comment

===============================================
@coursegrap-py 의 기본 YAML 형식

Default key (필수로 지정되어져야하는 키)
과목:
  - 학년: (int) (숫자만 입력) ex) 1
  - 학기: (int) (숫자만 입력) ex) 2
  - 과목명: (str) (한글, 영어 문자만 입력) ex) 웹스크립트프로그래밍

option key (옵션으로 지정되는 키) (옵션으로 지정되는 데이터의 경우 velue 값이 없다면 key를 지정하지 말 것)
  - 트랙:
    - (str) (한글, 영어 문자만 입력) 
  - 마이크로디그리:
    - (str) (한글, 영어 문자만 입력)
